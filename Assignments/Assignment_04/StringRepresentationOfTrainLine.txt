String representation of the TrainLine object should be the name/color of the
TrainLine, this means we need to add a variable to name the TrainLine. Before
Writing a toString() for TrainLine we should write one for Station first that
simply returns this.name for Station when printed. Once we have that write a 
toString() for TrainLine that will return the name of the TrainLine followed by
a list of every station in the line. To do this loop through each station and 
print it using the Station's toString().

PSUEDO:
in Station:
toString(){
    return this.name
}

in TrainLine:
toString:
String stations = this.name;
Station currentStation
while(){
    stations += currentStation;
    currentStation = currentStation.getNext();
}
return stations;
